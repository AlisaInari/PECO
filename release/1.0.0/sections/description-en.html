The Emission Conversion Factor Ontology: Description back to ToC

The next figure shows an overview of the PECO ontology. PECO represents traces of emission calculations, aligned with the W3C PROV standard.

An emission score is generted by an emission generation activity, which participates in an peco:emissionObservation.
